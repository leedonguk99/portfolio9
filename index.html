오늘의 추천
"당신에게 추천: 에티오피아 내추럴 (유사도 87%) — 산미·베리향 선호 반영"
(방법: 유사도 기반 필터링 + 콘텐츠 설명)
시도 권장
신규 블렌드 테스트 → 추천 반응이 좋을 시 캠페인 전개
(방법: 군집화 및 전환율 분석)
🤖 AI: “당신은 밝고 과일향의 커피를 선호합니다 — 오늘은 에티오피아 내추럴을 추천합니다.”
  1. 데이터 수집: 평가, 클릭, 재구매 데이터 수집
  2. 특성화: 로스팅·산지·향미를 표준 특성으로 변환
  3. 추천: 협업 필터링 + 콘텐츠 기반 혼합 모델 적용
  4. 설명: 추천 이유(특성 기여도)를 시각화

기술적 접근 (Tech)

데이터 정규화, 취향 벡터화, 하이브리드 추천 모델로 정확도와 설명력을 확보합니다.

기대 효과 (Impact)

사용자는 취향에 맞는 커피를 쉽게 발견하고, 로스터리는 개인화된 고객 경험으로 매출을 개선할 수 있습니다.
